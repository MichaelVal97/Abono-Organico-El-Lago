# **App Name**: MierdaCar Ecommerce

## Core Features:

- Product Listing: Display products from Firestore in a responsive grid layout on the homepage.
- Product Detail Page: Show detailed product information, including description, price, and stock.
- Add to Cart: Allow users to add products to a shopping cart.
- Shopping Cart: Display selected items with quantity adjustment and total calculation.
- User Authentication: Implement user registration and login using Firebase Auth.
- Firestore Integration: Read product data from Firestore database.
- Customer Feedback Summary: The app uses generative AI to summarize positive or negative customer feedback, including reasoning to back the claims. Functions as a tool to give the business owner insights to improve their offerings.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to align with the natural/agricultural theme.
- Background color: Light beige (#F5F5DC) for a clean and natural feel.
- Accent color: Brown (#A0522D) for buttons and highlights.
- Body and headline font: 'PT Sans' sans-serif for a balance of modern and friendly design. 
- Slogan Font: 'Alegreya' serif for an elegant intellectual touch.
- Use icons related to farming, nature, and agriculture. Simple, line-based icons are preferred.
- Responsive design with a grid layout for product listing (3 columns on desktop, 1 column on mobile).
